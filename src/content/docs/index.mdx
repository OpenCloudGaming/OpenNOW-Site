## Quick start

<Steps>
1. **Download** the latest release from [GitHub Releases](https://github.com/OpenCloudGaming/OpenNOW/releases).

2. **Install** the package for your platform:
   - **Windows** — NSIS installer or portable `.exe`
   - **macOS** — `.dmg` or `.zip` (Apple Silicon + Intel)
   - **Linux** — `.AppImage` or `.deb` (x64 + ARM64)

3. **Sign in** with your NVIDIA GeForce NOW account.

4. **Play** — browse games, launch a session, and tweak stream settings as you go.
</Steps>

## Features

| Category | What's included |
|----------|----------------|
| **Streaming** | WebRTC playback with codec selection (H.264, H.265, AV1), resolution up to 4K, FPS up to 360, bitrate cap, color quality modes, region preference, L4S support |
| **Authentication** | Browser-based OAuth + PKCE with persistent session state and automatic token refresh |
| **Game library** | Browse main catalog, personal library, and public game listings with search |
| **Input** | Pointer lock, keyboard, mouse (sensitivity + acceleration tuning), gamepad (up to 4), clipboard paste |
| **Microphone** | Disabled / push-to-talk / voice-activity modes with device selection and in-stream toggle |
| **Capture** | Screenshots (PNG/JPG/WebP) and video recording (MP4/WebM) saved locally with in-app gallery |
| **Diagnostics** | Live stats overlay — RTT, bitrate, packet loss, decode/render timing, input pressure, lag reason |
| **Shortcuts** | Fully configurable hotkeys for stats, pointer lock, stop session, anti-AFK, mic toggle, screenshot, and recording |
| **Controller mode** | Controller-first library UI with animated backgrounds, UI sounds, and auto-fullscreen options |
| **Cross-platform** | Windows (x64, ARM64), macOS (Apple Silicon, Intel), Linux (x64, ARM64) |

## Architecture at a glance

OpenNOW is an Electron app split into three process layers:

| Layer | Responsibility |
|-------|---------------|
| **Main** (Node.js) | OAuth, CloudMatch sessions, WebSocket signaling, settings, media storage |
| **Preload** | Typed `window.openNow` IPC bridge via `contextBridge` |
| **Renderer** (React) | UI, WebRTC peer connection, input capture, stats overlay, capture controls |

The active source lives in `opennow-stable/src/`. Shared types in `src/shared/gfn.ts` keep both sides in sync.
