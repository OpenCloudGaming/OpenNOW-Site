## Quick Start

<Steps>
1. **Download** the latest release from [GitHub Releases](https://github.com/zortos293/OpenNOW/releases)

2. **Install** the application
   - Windows: Run the installer or extract portable version
   - macOS: Mount DMG and drag to Applications
   - Linux: Install .deb/.rpm or use AppImage

3. **Login** with your NVIDIA GeForce NOW account

4. **Play** — Browse games and start streaming!
</Steps>

## Known Issues

<Aside type="danger" title="Performance Notice">
The app is being actively optimized. Report any issues on GitHub.
</Aside>

| Issue | Workaround |
|-------|------------|
| **High CPU usage** | Lower FPS/resolution. Optimization ongoing. |
| **Green screen flashes** | Switch to H.264 codec in settings. |
| **Audio stuttering** | Restart stream. Buffer improvements planned. |
| **Laggy input feeling** | Enable low-latency mode in settings. |
